# 递归函数的返回值

递归函数什么时候需要返回值？什么时候不需要返回值？这里总结如下三点：

1. 如果需要搜索整颗二叉树且不用处理递归返回值，递归函数就不要返回值。（113）
2. 如果需要搜索整颗二叉树且需要处理递归返回值，递归函数就需要返回值。（236）
3. 如果要搜索其中一条符合条件的路径，那么递归一定需要返回值，因为遇到符合条件的路径了就要及时返回。（112）
